🚀 Hosting on GitHub Pages

  1. Create a GitHub Repository:

    • Go to GitHub and log in.

    • Click on New to create a repository.

    • Name it something like yakov-resume.

    • Make sure the repository is public.

  2. Upload the HTML File:

    • On your new repo page, click on Add file → Upload files.

    • Upload the yakov-resume.html file (the one you copied).

    • Commit the changes.

  3. Enable GitHub Pages:

    • Go to the Settings tab of your repository.

    • Scroll down to the GitHub Pages section.

    • Under Source, select main branch and /root folder.

    • Click Save.

  4. Access the Hosted Resume:

    • After a few minutes, your resume will be hosted at:
      https://<your-username>.github.io/<your-repository-name>

🚀 Hosting on Netlify

  1. Sign Up / Log In to Netlify:

    • Go to Netlify, and sign up for an account or log in.

  2. Drag and Drop Deployment:

    • Once logged in, click on New Site from Git (or use Drag & Drop).

    • Upload the yakov-resume.html file directly.

  3. Configure Settings:

    • Once uploaded, Netlify will give you a live URL, and your site will be live almost instantly.

Both methods are free and easy to set up. The GitHub Pages method is perfect if you're familiar with version control and want to keep your projects updated. Netlify offers more options for continuous deployment and custom domains, but both work great for this kind of simple static page!
